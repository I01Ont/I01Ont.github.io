使用 Github 搭建基于 Hexo 的 Blog

最近用 Github 搭建了一个基于 Hexo 的 Blog，是在 Mac OS X 上完成的，在其它平台的都是大同小异的。至于什么是 Github 和 Hexo 在此就不再作过多介绍了，如不曾接触过，可先行了解。

至于是否会写一篇搭建过程的文章，这会在以后写出来，先在这里给出参考过的文章：

  • Hexo在github上构建免费的Web应用

  • 手把手教你建github技术博客by hexo

  • Hexo(一):在GitHub上搭建静态博客

我用的主题是 Next。
比较容易出现的问题，例如忘了安装 git 、打包上传到 github 时的地址出错 、域名解析的问题等等，只要仔细耐心去做都是能解决的。

现在你是否也想动手去搭建一个属于自己的 Blog ,那就别犹豫了，喜欢就去做。
